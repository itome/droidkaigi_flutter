## Getting Started

Setup your flutter environment:
https://flutter.dev/docs/get-started/install

Enalbe web support

```
$ flutter channel beta
$ flutter upgrade
$ flutter config --enable-web
```

Generate auto generated files
```
$ flutter pub run build_runner build
```

Run app
```
$ flutter run
```
